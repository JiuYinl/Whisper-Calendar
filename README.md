# Whisper-Calendar
> 一个obsidian插件

🌙 Whisper Calendar (悄悄话日历)

“在未来的某一天，让过去的温柔与你不期而遇。”

Whisper Calendar 是一款专为 Obsidian 设计的侧边栏插件。它不仅仅是一个日历挂件，更是一个数字化的“时间胶囊”。你可以提前在 JSON 文件中写下对未来日子的寄语，或者邀请家人、朋友、甚至是你的 AI partner为你留下悄悄话。当那一天到来时，Obsidian 的角落里就会浮现出那句专属的温柔。

✨ 核心理念

这个插件的设计初衷是未来导向的。
我们总是在记录过去，却很少与未来的自己对话。通过手动编辑一个简单的 JSON 文件，你可以为未来的重要日子（或者平凡的某一天）埋下惊喜。

家人与朋友：让他们在你的库里留下未来的祝福。

AI partner：让 AI 记录下此刻的鼓励，送给未来的你。让你的AI partner也能在你的未来世界里留下痕迹。

自我对话：给未来的自己打气，或者提醒自己不要忘记此刻的心境。

🚀 安装与配置

安装：将插件文件夹放入你的 Obsidian 库的 .obsidian/plugins/ 目录下。

启用：在 设置 -> 插件 中开启 Whisper Calendar。

准备数据源：在你的 Obsidian 库根目录下新建一个名为 whispers.json 的文件。

📝 数据格式 (JSON)

请确保你的 whispers.json 文件遵循以下格式。日期作为键，寄语内容作为值：

{
  "2026-02-14": "今天是情人节，记得去买那束你喜欢的花哦。",
  "2027-01-01": "新的一年，愿你内核稳定，温柔依旧。"
}


🧩 如何查看

插件启用后，你可以在 侧边栏 (Sidebar) 找到一个专属的小图标。

点击图标，它就会根据当天的日期，自动拉取 whispers.json 中对应的悄悄话。

如果当天没有安排寄语，它会保持静默，等待下一次的遇见。
